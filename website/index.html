D&D Beyond Kit

Available Stores

Install D&D Beyond Kit on your preferred browser:

Chrome Web Store Mozilla Firefox

Explore the code on GitHub.

Features

  • Translate D&D Beyond character sheets to your language.
  • Display metrics using the International Unit System (SI).
  • Streamline the character sheet interface, reducing visual clutter.
  • Add a "Skip to the top" button for long pages.
  • Enable autoload on listing pages like monsters and spells.

This extension simplifies the character sheet for users who need translations and easier navigation.

Supported Languages

  • Brazilian Portuguese (Português)
  • Spanish (Español)
  • English (Metric System changes only)
  • Italian (Italiano)
  • French (Français)
  • German (Deutsch)
  • Czech (Čeština)
  • Japanese (日本語)
  • Dutch (Nederlands)
  • Russian (Русский)
